## 🧮 Calculator App

A simple and responsive **Calculator Web App** built using **HTML, CSS, and JavaScript**, capable of performing basic arithmetic operations with a clean user interface.

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox)
* JavaScript (DOM Manipulation)

## ⚙️ Functionality

* User inputs numbers and operators using buttons
* Expression is displayed on screen
* Result is calculated instantly on pressing `=`

## 💡 Learnings

* DOM manipulation using JavaScript
* Handling user input and events
* Building responsive layouts using CSS
* Managing calculation logic and edge cases
